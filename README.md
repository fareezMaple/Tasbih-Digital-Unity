# Tasbih-Digital-Unity
Counter like app. Made with Unity. Code in c#

Feature of this app:
1) Count number with increment of 1
2) Sound feedback*
3) Vibration feedback*
4) Save progress when exiting the app
5) Potrait and landscape orientation
